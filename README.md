# Chat Room 
A slack like chat room that allows you to create channels to discuss about things one cares about

## Tech Stack
* React
* Firebase
* Styled-Component
* React-Router
* React-Hooks
* React-Redux

## Get Started
1. Open 'src/firebase.js' file and enter firebase credentials
2. Go to project directory and run:
```npm install```
3. Start The project 
```npm start```
